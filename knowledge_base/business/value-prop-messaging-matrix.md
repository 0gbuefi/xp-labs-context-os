# Value Prop: Messaging Matrix by Audience Segment

## What This Is

A ready-to-use messaging framework with primary message, supporting points, and recommended CTA for each of XP Labs's 7 target audience segments. Use this as the reference when crafting cold emails for a specific ICP.

## Segment-Specific Primary Messages

| Segment | Primary Message |
|---------|----------------|
| **Copywriters** | Find the exact words your customers use, and turn them into copy that converts |
| **Founders** | Write marketing copy that sounds like your customers, not like a founder guessing |
| **AI Writers** | Feed your AI the real words humans use, not more AI-generated phrasing |
| **Content Writers** | Write content that uses the same words your readers use when they talk about this topic |
| **SaaS Marketers** | Stop A/B testing headlines you guessed. Start with the words your users already say |
| **Health Copywriters** | Find how real people describe their health frustrations, in their own words, not clinical language |
| **E-commerce Brands** | Find the words shoppers use when they love (or hate) products like yours |

## How This Informs Campaigns

This matrix is the starting point for every campaign idea. When generating cold email angles:
1. Identify the target ICP
2. Look up their primary message
3. Pick 2-3 supporting points that match the persona
4. Use the recommended CTA
5. Combine with a relevant pain point and front-end offer

The matrix ensures messaging is audience-specific rather than one-size-fits-all. Different ICPs need different lead messages even though the product is the same.
