# Product: Bookmarks

## What It Does

As you review themes and excerpts, bookmark the phrases that jump out. Star a bookmark to mark it as a top pick. Add notes to bookmarked excerpts — tag for a specific copy project or jot down how you plan to use a phrase. Bookmarks page collects excerpts across all sessions into a personal swipe file.

Bookmarks are workspace-scoped — everyone on the team sees the same bookmarked excerpts.

## Key Campaign Messaging

- "Not every excerpt is gold. Bookmarks let you curate the best of the best."
- "When you start a new project, check your bookmarks first. You might already have the perfect phrases."
- "A shared swipe file that grows across your whole team's research."

## Where It Appears

- Excerpt cards: bookmark icon on every excerpt in Notes tab
- Bookmarks page (`/bookmarks`): all bookmarked excerpts across sessions
- Session detail: bookmarked excerpts tab within each session
