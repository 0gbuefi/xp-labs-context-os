# Product: AI Theme Extraction

## What It Does

After listening to Reddit threads, XP Labs analyzes all conversations and groups them into themes: recurring patterns in how people talk about a topic. Each theme contains verbatim excerpts tagged by research type (Pain, Dream, Obstacle, Identity, Trigger, Objection, Value Proposition) and awareness stage.

Overlapping themes merge automatically. New analysis rounds merge into existing theme sets.

## Key Campaign Messaging

- "Themes tell you WHAT to talk about. Excerpts give you the EXACT WORDS to use."
- "A single theme might have dozens of excerpts, each phrased differently by a real person."
- "The analysis step that turns raw Reddit threads into organized, copy-ready language."
