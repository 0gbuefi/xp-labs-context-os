# Product: Voice Profile Generation

**Status: Coming soon.** Built but not yet enabled — button visible in UI but returns "Coming Soon."

## What It Will Do

A voice profile is a synthesized reference document capturing recurring language patterns, tone, and vocabulary from research. Identifies key phrases, emotional triggers, sentence structures, word preferences, and specific metaphors. Formatted as readable prose — a cheat sheet for writing in your audience's natural voice.

## Key Campaign Messaging

- "A cheat sheet for writing in your audience's natural voice"
- "Bridges the gap between raw research and finished copy"
- "Sits open next to your copy draft as a writing reference"

**Note:** Since this is "coming soon," use carefully in campaigns. Can mention as upcoming feature but don't position as primary value prop.
