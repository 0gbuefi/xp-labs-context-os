# Weekly Send Timing Framework

Align cold email sends to natural weekly engagement patterns: **Monday for launches**, **Wednesday for peak engagement**, **Friday for auto-reply triage**. Timing compounds with copy quality - teams respecting these windows see measurably higher reply rates.

## Core Principles

- **Monday = Launch Day** - Fresh inboxes, clearer priorities, highest send volume
- **Wednesday = Peak Engagement** - Highest engagement metrics, ideal for persuasive follow-ups
- **Friday = Auto-Reply Surge** - OOO messages peak, triage opportunity for Monday restart
- **Timing matters as much as copy** - proper rhythm compounds results
- **3-4 day spacing** between touches maintains momentum without overwhelming
- **Natural readiness alignment** - catch prospects at different engagement moments

## When to Use

Use this weekly framework when:
- Running ongoing cold email campaigns
- Want to maximize engagement without trial-and-error
- Planning sequence timing across multiple touches
- Setting automation rules for send schedules
- Coordinating team campaign calendar
- Need consistent predictable rhythm

## How to Apply

### Weekly Cadence Framework

**Monday: Launch New Sequences**
- **Why:** Prospects return with fresh inboxes and clearer priorities
- **Action:** Launch new campaigns, start sequences
- **Volume:** Highest send volume across platform
- **Mindset:** Plant seeds for the week ahead
- **Avoid:** Don't bury launches later in week when attention fades

**Wednesday: Send Follow-Up Touches**
- **Why:** Prospects settled into week but haven't wound down for weekend
- **Action:** Send Step 2-4 follow-ups, persuasive angles
- **Performance:** Consistently highest engagement metrics
- **Timing:** 3-4 days after Monday launch = perfect Wednesday arrival
- **Strategy:** Save best follow-up copy for Wednesday

**Friday: Auto-Triage Responses**
- **Why:** Highest OOO auto-reply volume as prospects prep for weekend
- **Action:** Use automation to detect OOO, reschedule Monday follow-up
- **Review:** Campaign performance analysis, messaging pivots
- **Setup:** Configure subsequences triggered by "Out of office" keywords
- **Benefit:** Maintain sequence momentum without manual intervention

### Practical Implementation

**Campaign Launch Schedule:**
```
Week 1:
- Monday: Launch Sequence A (1,000 contacts)
- Wednesday: (waiting for responses)
- Friday: Triage OOO, adjust messaging

Week 2:
- Monday: Launch Sequence B (1,000 contacts)
- Wednesday: Send Step 2 for Sequence A (~600 remaining)
- Friday: Review Sequence A performance, triage OOO

Week 3:
- Monday: Launch Sequence C (1,000 contacts)
- Wednesday: Send Step 2 for Sequence B + Step 3 for Sequence A
- Friday: Campaign performance review, messaging adjustments
```

### Automation Setup

**OOO Detection (Friday triage):**
```
IF reply contains keywords:
- "Out of office"
- "On vacation"
- "Away from desk"
- "Will return"

THEN:
- Remove from current sequence
- Add to subsequence
- Resume outreach Monday (3-4 days later)
```

**Benefit:** Conversations restart on Monday when prospects return, maintaining natural rhythm.

### Best Day Analysis (Data-Driven)

**Monday:**
- Highest send volume industry-wide
- Fresh inbox = attention available
- Ideal for first-touch emails
- Sets foundation for week

**Tuesday:**
- Second-highest volume
- Still strong engagement
- Acceptable alternative to Monday

**Wednesday:**
- **Peak reply rates** (highest of week)
- Prospects fully engaged in work
- Haven't started weekend wind-down
- Best day for persuasive follow-ups

**Thursday:**
- Engagement starts declining
- Still acceptable for sends
- Better than Friday

**Friday:**
- **Lowest engagement** for campaign sends
- **Highest OOO auto-replies**
- Best for: Review, triage, analysis
- Worst for: New launches, important follow-ups

### Sequence Spacing Example

**4-email sequence with optimal timing:**

```
Touch 1: Monday, Week 1 (Launch)
Touch 2: Wednesday, Week 1 (3 days later - peak engagement)
Touch 3: Monday, Week 2 (5 days later - fresh start)
Touch 4: Wednesday, Week 2 (3 days later - final push)
```

**7-email sequence with optimal timing:**

```
Touch 1: Monday, Week 1 (Launch)
Touch 2: Wednesday, Week 1 (3 days - peak)
Touch 3: Monday, Week 2 (5 days - fresh angle)
Touch 4: Wednesday, Week 2 (3 days - peak)
Touch 5: Monday, Week 3 (5 days - persist)
Touch 6: Wednesday, Week 3 (3 days - peak)
Touch 7: Friday, Week 3 (3 days - breakup)
```

**Pattern:** Alternate between Monday launches and Wednesday peak engagement days.

### Team Calendar Coordination

**Agency managing multiple clients:**

**Monday AM:**
- Launch Client A campaigns
- Review weekend triage results
- Set week's priorities

**Wednesday AM:**
- Send all client follow-ups (Step 2-4)
- Highest reply volume expected
- Team preps for response surge

**Friday PM:**
- Performance review across all clients
- Identify campaigns needing pivots
- Triage OOO responses
- Plan next week's launches

**Benefit:** Predictable rhythm enables efficient team coordination.
