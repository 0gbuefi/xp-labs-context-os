# Six Core Offers Framework (Four for B2B)

There are only **6 universal offers** in the world. For B2B, focus on the top 4: **save time, save money, make money, reduce risk**. Every campaign should clearly communicate which benefit you deliver.

## Core Principles

- All offers reduce to these 6 categories (drop "raise status" and "live longer" for B2B)
- Most businesses can message all four B2B offers
- Test which offer resonates most with your ICP
- Use this as initial roadmap for campaign ideation
- Change offer angle between email sequences for variation

## When to Use

Use this framework when:
- Ideating initial campaign variations (run 3-4 tests)
- Campaign isn't working and needs pivot
- Testing different value props in Email 1 variants (A/B/C)
- Analyzing why competitor messaging works
- Explaining value prop to stakeholders clearly

## How to Apply

**For Campaign Ideation:**

Take your core service and map to all four offers:

**Example: Cold Email Agency (GEX)**
- **Save Time:** "We already know how all the tools work - no training needed"
- **Save Money:** "Half the cost of hiring SDR team"
- **Make Money:** "Generate more pipeline and close more deals"
- **Reduce Risk:** "Free test campaign before committing"

Create separate email sequences for each angle, test which resonates best.

**For Variant Testing Within Email 1:**

Test different offer angles across Email 1 variants (A/B/C):

**Variant A (Make Money):** Lead with revenue/growth angle
```
"We've helped companies like Prospeo build over $3M in pipeline..."
```

**Variant B (Save Money):** Lead with cost reduction angle
```
"Our outbound service is half the cost of an SDR and generates the same pipeline..."
```

**Variant C (Save Time):** Lead with efficiency angle
```
"We handle all the outbound work so you can focus on closing deals..."
```

**Example: Testing Offer Angles (GEX Templates)**

All three variants share:
- Same targeting (new VP of Sales, no SDR team)
- Same structure (four sentences)
- Same case study (Prospeo $3M pipeline)
- Different offer angle in Sentence 2

**Make More Money Variant:**
"Do you have a plan to build pipeline with such a lean team?"

**Save Money Variant:**
"Have you thought about outbound automation so you can save money from hiring a team of SDRs?"

**Save Time Variant:**
"Do you have any outbound systems running yet or are you looking to get started without sinking tons of time?"

By changing just one sentence across variants, create 3 distinct offer tests within a single campaign.

## Evidence

[VERIFIED: GEX Wrapped 2024] "There are only 6 offers in the world. You can help people: Save time, Save money, Make more money, Reduce Risk, Raise their status, Live longer. In a B2B context, we can throw away raise status and live longer and stay focused on the first four."

[DATA: Campaign variations] "If you use this framework combined with changing around how you communicate your offers, you can easily get 3-7 campaign tests going just shaking up how you communicate saving time, saving money, and making more money to your customers."

[VERIFIED: Practical application] "Most businesses can have messaging into all four of the top B2B offers. As a cold email agency, we help people save money because we can do the same level of activity as a multi person SDR team. We also help people make more money by getting leads. We help them save time because we already know how all of the tools work."
