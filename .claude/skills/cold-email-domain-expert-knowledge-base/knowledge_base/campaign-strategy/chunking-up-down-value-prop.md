# Chunking Up & Down on Value Prop

When initial campaign messaging doesn't work, **chunk up** to broader strategic outcomes or **chunk down** to specific tactical features. This varies the level of abstraction to find what prospects care about most.

## Core Principles

- Start with mid-level value prop (e.g., "we create pipeline")
- If no response, test strategic level (broader outcomes)
- Also test tactical level (specific features)
- Most products can message at 3+ levels of abstraction
- Test reveals which level resonates with ICP

## When to Use

Use chunking when:
- Initial value prop messaging falls flat
- Prospects don't respond to feature lists
- Offer seems commoditized
- Need to differentiate from competitors saying same thing
- Want to generate 3-5 campaign variations quickly

## How to Apply

**Example: Cold Email Agency (GEX)**

**Mid-Level Value Prop (Starting Point):**
"We create more pipeline for B2B companies with cold email"

**Chunk UP (Strategic Outcomes):**
- "No need to train SDRs anymore just for cold email"
- "Build pipeline without adding headcount"
- "Scale lead generation without hiring"
- Get away from features entirely, speak only about outcomes

**Chunk DOWN (Tactical Features):**
- "We manage your email deliverability"
- "No expensive long-term contracts with data providers"
- "Proven copywriting methodology already built"
- "AI-generated emails that humans can't detect"

Focus on little things you handle that solve specific problems.

**The Pattern:**

```
STRATEGIC (Chunk Up)
    ↑
    "What broader outcome does this enable?"

MID-LEVEL (Starting Point)
"We create pipeline with cold email"

    ↓
    "What specific things do we do to deliver this?"

TACTICAL (Chunk Down)
```

**Process:**
1. Write your current value prop
2. Ask: "What broader outcome does this create?" (chunk up)
3. Ask: "What specific features deliver this?" (chunk down)
4. Create email variations at each level
5. Test all three, measure responses
6. Double down on winning abstraction level

**Example Application: CRM Pitch**

**Chunk Up:**
"Free up your sales team to focus on selling instead of data entry"

**Mid-Level:**
"Sales CRM with built-in data enrichment"

**Chunk Down:**
"LinkedIn Chrome extension that syncs conversations to CRM automatically"

Often the chunked-down version (specific pain point) outperforms generic mid-level pitch.
